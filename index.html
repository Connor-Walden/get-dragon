Get Started

This CSS framework started as a joke made during my web development bootcamp... And it turned into what it is now! This framework was written in 100% vanilla CSS and is cross-browser compatible. This framework uses a grid system and all components try to stretch to fit the column they are in. There are many features in the grid that allow offsetting columns by however many column widths you want and a dock class to make sure a column is stuck to the left of the row.

When developing this framework a user-experience first approach was taken and all Dragon classes are designed to be memorised easily. This framework supports not only rows and columns but sub-rows and sub-columns meaning you can continue to break the page up into smaller and smaller grids for easy positioning.
